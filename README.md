# This Day in Space History

**Automatisk hver dag viser siden:**

1. **Astronomy Picture of the Day (APOD)**  
   Stort bilde + original NASA-tekst skrevet av astronomer.

2. **Alle tidligere APOD-bilder på akkurat denne datoen (1995 → i dag)**  
   Et grid med alle bilder fra samme kalenderdag gjennom 30 år.  
   Hvert bilde har:
   - Årstall
   - Original tittel
   - Klikk for å se full forklaring
